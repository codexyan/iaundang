# LOGO GENERATION PROMPT - AKUNDANG

## 📋 PROMPT UNTUK CHATGPT/DALL-E

### **PROMPT VERSI 1 - DETAILED (Recommended)**

```
Create an elegant, minimalist logo for "Akundang" - a premium digital wedding invitation platform for modern Indonesian couples.

BRAND IDENTITY:
- Name: Akundang (combining "aku" = me/us + "undang" = invite)
- Industry: Digital wedding invitations SaaS
- Target: Young couples aged 22-35, modern & tech-savvy
- Positioning: Premium, elegant, sophisticated yet approachable

VISUAL STYLE:
- Design: Elegant, simple, clean, modern minimalist
- Mood: Romantic, premium, trustworthy, celebratory
- Style: Contemporary with timeless appeal

COLOR PALETTE (MUST USE):
Primary Colors:
- Forest Green: #1a3320, #2d5a3d (deep elegant green)
- Gold: #b8860b, #d4af37 (premium gold)
- Combination: Green-to-gold gradient for depth

Supporting:
- White for contrast
- Champagne gold #c9952d as accent

DESIGN ELEMENTS TO CONSIDER:
Symbol ideas (choose ONE or combine subtly):
- Heart shape (love, wedding)
- Ring/circle (unity, eternal bond, wedding ring)
- Two elements coming together (couple unity)
- Envelope (invitation concept)
- Letter "A" stylized elegantly
- Abstract celebration symbol

Typography:
- Modern serif or elegant sans-serif
- Clean, readable letterforms
- "ak" could be emphasized (brand name starts with it)
- Optional: "akundang" wordmark alongside symbol

TECHNICAL REQUIREMENTS:
- Format: Flat design, vector-style
- Scalability: Works at small (16px) and large sizes
- Versatility: Looks good on white background AND dark forest green background
- Simplicity: Maximum 2-3 colors, minimal details
- Icon: Should work standalone without text

INSPIRATION KEYWORDS:
Elegant, minimal, modern, premium wedding, luxury invitation, sophisticated romance, contemporary Indonesian design, clean geometric, refined simplicity

OUTPUT:
Create 3 logo variations:
1. Icon + Wordmark (full logo)
2. Icon only (app icon / favicon)
3. Wordmark only (alternative usage)

AVOID:
- Overly decorative or busy designs
- Cheap clipart style
- Generic wedding clichés (doves, bells)
- Too many colors or gradients
- Difficult to reproduce elements
```

## 📝 TIPS REVISI

Kalau hasil kurang sesuai, minta revisi dengan:

**Untuk Style:**
- "Make it more minimal and geometric"
- "Simplify the icon, remove details"
- "Make it more elegant and premium"
- "Use cleaner lines, less ornamental"

**Untuk Color:**
- "Use darker forest green #1a3320"
- "Add gold accent #d4af37"
- "Make green-to-gold gradient smoother"
- "Reduce to 2 colors only"

**Untuk Symbol:**
- "Make the heart more subtle and abstract"
- "Simplify the ring shape"
- "Focus on lettermark instead of icon"
- "Combine letter A with heart concept"

**Untuk Typography:**
- "Use more elegant serif font"
- "Make wordmark more prominent"
- "Adjust letter spacing for readability"
- "Try modern sans-serif instead"

## 🎯 BEST PRACTICES

1. **Start Simple**: Mulai dengan prompt pendek, lalu minta variasi
2. **Iterate**: Generate 3-5 versi, pilih yang terbaik, lalu refine
3. **Test Scale**: Pastikan logo terlihat bagus di ukuran kecil (favicon)
4. **Color Test**: Cek logo di background putih DAN hijau gelap
5. **Save Variations**: Simpan versi icon-only, wordmark-only, dan full logo
